# Zihao-Feng
商品销售数据分析项目
项目背景
      本项目基于电商行业快速发展的背景，针对运动服饰店铺的销售数据进行深度分析。随着线上购物竞争日益激烈，商家需要通过数据驱动的方式优化运营策略。本项目利用Python技术栈对商品销售数据进行多维度分析，旨在为店铺的商品管理、库存规划和营销策略提供数据支持，帮助商家在竞争激烈的市场环境中提升竞争力。

数据分析方法与结论
       本项目采用完整的数据分析流程：首先使用pandas进行数据清洗和合并，处理商品信息与销售记录的关联；然后从时间维度分析月度销售趋势，从品类维度分析商品大类和子类的销售贡献；最后通过matplotlib可视化展示分析结果。分析发现销售呈现明显季节性特征，夏季为销售高峰，运动上装类商品贡献主要销售额，其中背心和短袖子类表现突出。基于这些发现，建议商家根据销售季节性调整库存，加强畅销品类的深度开发，并制定差异化的促销策略。
